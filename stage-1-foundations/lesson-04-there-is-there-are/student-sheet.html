Lesson 4 — There is / There are

In this lesson you learn to write about rooms and places: what is in them and where things are. You will use there is / there are and small words like in, on, under, next to. Read each example first, then write your own answers.

Key language

there is (one thing) / there are (more than one thing) PT: "there is / there are" = há / tem.

  • There is a table. (one) → singular
  • There are two chairs. (more than one) → plural
  • Careful: plural always uses there ARE. Not "there is two chairs".

Negative: there isn't / there aren't

  • There isn't a sofa.
  • There aren't any windows.

Questions: Is there…? / Are there…?

  • Is there a lamp? — Yes, there is. / No, there isn't.
  • Are there any chairs? — Yes, there are. / No, there aren't.

a / an / some / any PT: usamos a/an com uma coisa, some/any com várias.

  • There is a door. / There is an apple.
  • There are some chairs. / There aren't any windows.

Prepositions of place (where things are) PT: dizem onde está a coisa.

  • in (em/dentro), on (em cima de), under (debaixo de), next to (ao lado de), between (entre), behind (atrás de), in front of (em frente de), near (perto de)

Room & furniture words: table (mesa), chair (cadeira), sofa (sofá), bed (cama), window (janela), door (porta), kitchen (cozinha), bathroom (banheiro), wall (parede), lamp (luminária).

(b) Choose is / are or isn't / aren't

Example: There ___ two doors. → There are two doors.

  1. There ______ (is / are) one bed in the room.
  2. There ______ (is / are) two lamps on the table.
  3. There ______ (isn't / aren't) a sofa in the kitchen.
  4. There ______ (isn't / aren't) any chairs in the bathroom.
  5. There ______ (is / are) a window near the door.
  6. There ______ (is / are) three chairs next to the table.

(c) Fill in the correct preposition

Word bank: in · on · under · next to · between · behind · in front of · near

Example: The lamp is ___ the table. → The lamp is on the table.

  1. The chair is __________ the table. (em frente de)
  2. The bed is __________ the wall. (perto de)
  3. The shoes are __________ the bed. (debaixo de)
  4. The sofa is __________ the two windows. (entre)
  5. The door is __________ the wall. (atrás de)
  6. The lamp is __________ the bed. (ao lado de)

Homework

Write 5 sentences describing a kitchen. Use there is / there are in every sentence, and use a preposition from the word bank below in at least three of them.

Word bank: in · on · under · next to · between · near · table · chair · window · door · wall · lamp

Example to start you off: There is a table in the kitchen.
